# SSI
This project is project spring boot version 2.0.0,
the following tools are need to start working:
* Java version 1.8 or higher
* Maven version 3.5.3 or higher

# Modules and Libraries
The libraries:
* Spring Data JPA
* Jackson 
* Swagger2
